# 📁 Dataset Information

## Superstore Sales Dataset

- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/vivek468/superstore-dataset-final
- **File Name:** Sample - Superstore.csv
- **Size:** ~2.4 MB
- **Rows:** 9,994
- **Columns:** 21

## Columns Overview

| Column | Description |
|--------|-------------|
| Order ID | Unique order identifier |
| Order Date | Date of order placed |
| Ship Date | Date of shipment |
| Ship Mode | Shipping method |
| Customer ID | Unique customer identifier |
| Customer Name | Name of customer |
| Segment | Customer segment (Consumer / Corporate / Home Office) |
| Country | Country of sale |
| City | City of sale |
| State | State of sale |
| Region | Region (East / West / Central / South) |
| Product ID | Unique product identifier |
| Category | Product category (Technology / Furniture / Office Supplies) |
| Sub-Category | Product sub-category |
| Product Name | Name of the product |
| Sales | Revenue from the order |
| Quantity | Units ordered |
| Discount | Discount applied |
| Profit | Profit from the order |

## How to Use

Download the dataset from Kaggle and place it in your working directory, then update the file path in Cell 2 and Cell 4 of the notebook accordingly.
